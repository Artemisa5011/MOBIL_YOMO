⸸ YOMI NO HANA — Aplicación Móvil ⸸

Yomi No Hana es la versión móvil del sistema de gestión del Templo Fúnebre. Permite a vendedores y clientes gestionar servicios funerarios y reservas de lotes de cementerio, todo conectado a la misma base de datos Supabase del proyecto web.

Problema que resuelve: los vendedores necesitaban acceso móvil al sistema para registrar clientes y cerrar ventas de servicios fúnebres y lotes desde cualquier lugar, sin depender de una computadora.
